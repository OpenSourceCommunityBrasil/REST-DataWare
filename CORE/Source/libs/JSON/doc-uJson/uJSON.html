Descrição

Contém todas as classes para manipular o formato JSON.

As principais classes são : TJSONObject, TJSONArray .

OBS: A unit uJSON foi adaptada de uma implementação para json em Java ver http://www.json.org .

JSON = JavaScript Object Notation

Utilizando:
para criar um objeto json a partir da string:

s := '{"Commands":[{"Command":"NomeComando", params:{param1:1}}]}' ;
json := TJSONObject.create (s);

para acessar os elementos:
json.getJSONArray ('Commands) // retorna uma TJSONArray dos comandos

para retornar a string "NomeComando":
json.getJSONArray ('Commands).getJSONObject(0).getString('Command')
